В приложенных файлах находится клиент-серверное приложение по созданию заметок и управления ими.
Реализуй:
1. Добавление заметки в базу данных
2. Удаление заметок из базы данных
3. Редактирование заметок

Для добавления используй существующий код сайта, дополни его запросами на сервер с добавлением заметки на сервере в базу данных.

Для удаления используй существующий код сайта, дополни его запросами на сервер с удалением конкретной заметки на сервере из базы данных.

Для редактирования используй существующий код сайта, дополни его запросами на сервер с изменением конкретной заметки на сервере.
